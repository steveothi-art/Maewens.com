# ✨ MAEWEN - Agence Digitale

## Lumière • Harmonie • Prospérité

MAEWEN est une agence créative spécialisée dans la **création graphique**, le **développement web**, les **sites vitrines**, les **solutions e-commerce** et le **branding digital**.

Nous concevons des expériences digitales modernes, élégantes et performantes pour aider les entreprises à se démarquer en ligne.

---

## 🚀 Aperçu du projet

Ce projet est un site web vitrine moderne comprenant :

- Un **preloader animé**
- Un **hero section impactant**
- Une présentation des **services**
- Une section **à propos**
- Une **méthode de travail**
- Une page **contact avec WhatsApp**
- Un **design premium orienté UX/UI**

---

## 🎯 Objectif

Créer une présence digitale forte pour MAEWEN en mettant en avant :

- La qualité du design
- L'expérience utilisateur
- La performance web
- L'identité visuelle de l'agence

---
